## Mini Proyecto: HelloKittyOnline
Crear una página web de ciberseguiridad capaz de registrarlas y clasificarlas de manera clara y ordenada, así como ofrecer información acerca de las mismas que pueda resultar útil para los usuarios. Dicha información deberá incluir: cuál es la amenaza, su clasificación, nivel de riesgo y acciones recomendadas. 

## Estructura: 
- pages: Código.
- Data: Datos en crudo (raw) y en limpio (clean).

## Instrucciones:
- Virtual env: python -m venv .venv
- Activarlo: .venv\Scripts\activate
- Librerías: pip install -r requirements.txt

## Porque pages:
- Al usar ese nombre streamlit automaticamente coloca un sidebar por el funcionamiento basico de la libreria, eso esta hecho de manera predeterminada y se configura a traves del st.sidebar.
